Next steps

  • Create a custom checkout session with the CLI:

    $ stripe checkout sessions create \
    --success-url="https://example.com/success" \
    --cancel-url="https://example.com/cancel" \
    -d "payment_method_types[]"=card \
    -d "line_items[0][name]"="Stripe Press" \
    -d "line_items[0][description]"="Get Together" \
    -d "line_items[0][amount]"=2300 \
    -d "line_items[0][currency]"=usd \
    -d "line_items[0][quantity]"=2 \
  | stripe-dev checkout run

  • Clone one of our checkout samples with the CLI to see more indepth use cases

    • checkout-one-time-payments
    • checkout-single-subscription
    • checkout-subscription-and-add-on

  • Make sure to configure a webhooks endpoint for checkout.session.completed and use that to a completed payment.
